# Function: CheckForWinVistaErrorDialog

## Description

Use this function in WindowCreatedEvent to check for Windows Vista error
dialogs, and if so it will schedule focus to move to the error dialog.

## Returns

Type: Void\

## Parameters

### Param 1:

Type: handle\
Description: The handle of the window which will be check if it belongs
to the Vista error dialog.\
Include: Required\

## Version

This function is available in the following releases:

1.  JAWS 10.0 and later
