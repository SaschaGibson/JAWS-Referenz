# []{#_JAWS5.10-21.0} Function: SetSystemVolume

## Description

Sets the system volume.

## Returns

Type: Void\

## Parameters

### Param 1:

Type: Int\
Description: The new value to which you wish to set the volume. Volume
of 0 percent to 100 percent ranges in value from 0 to 0xffff (65535).\
Include: Required\
