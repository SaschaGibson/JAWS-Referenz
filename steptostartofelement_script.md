# Script: StepToStartOfElement

## Description

Moves backward and up one level in the element hierarchy. This means you
will be taken to the beginning of the current table, list, or nested
list. From there you will be taken to the next level up.

## Returns

Type: Void\

## Parameters

No Parameters

## Version

This function is available in the following releases:

1.  JAWS 4.51 and later
