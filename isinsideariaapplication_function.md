# Function: IsInsideARIAApplication

## Description

Used to determine if the virtual PC cursor is inside an ARIA application
on a web page. An ARIA application is generated by the
role=\"application\" attribute on an HTML element.

## Returns

Type: Int\
Description: 1 (True) or 0 (false).\

## Parameters

No Parameters

## Version

This function is available in the following releases:

1.  JAWS 14.0 and later
