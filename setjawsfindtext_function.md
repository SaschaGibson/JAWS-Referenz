# Function: SetJAWSFindText

## Description

Sets the text to be located when JAWSFindNext runs.

## Returns

Type: Void\

## Parameters

### Param 1:

Type: string\
Description: The text to be used with JAWSFindNext.\
Include: Required\

## Version

This function is available in the following releases:

1.  JAWS 6.10 and later
