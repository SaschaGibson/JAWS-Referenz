# Function: ShouldSayAllOnDocumentLoad

## Description

Determines if user settings indicate that SayAll should be started after
a web page or document loads.

## Returns

Type: int\
Description: True if SayAll should be started after a web page or
document loads, false otherwise.\

## Parameters

No Parameters

## Version

This function is available in the following releases:

1.  JAWS 18.0 and later
