# Function: GetBrailleDisplayName

## Description

Returns the name of the braille display that is currently in use.

## Returns

Type: String\
Description: The name of the braille display that is currently in use.
If no braille display is attached, this function returns an empty
string.\

## Parameters

No Parameters

## Version

This function is available in the following releases:

1.  JAWS 9.0 and later
