# Script: TouchGoToPriorStatusBar

## Description

Moves to the prior status bar element with the Touch cursor. If no prior
status bar was found, the Touch cursor does not move and a message is
spoken indicating that there are no previous status bars.

## Returns

Type: Void\

## Parameters

No Parameters

## Version

This function is available in the following releases:

1.  JAWS 15.0 and later
