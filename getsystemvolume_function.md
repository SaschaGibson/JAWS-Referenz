# []{#_JAWS5.10-21.0} Function: GetSystemVolume

## Description

Gets the current system volume setting.

## Returns

Type: Int\
Description: The current system volume setting. Volume of 0 percent to
100 percent ranges in value from 0 to 0xffff (65535).\

## Parameters

No Parameters
