# []{#_JAWS4.51-14.0} Function: GetCurrentScriptKeyName

## Description

Retrieves the name of the key used to invoke the currently active
script.

## Returns

Type: String\
Description: The key name.\

## Parameters

No Parameters

## Version

This function is available in the following releases:
