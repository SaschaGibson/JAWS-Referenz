# Function: BrailleGetSplitMode

## Description

This function allows you to know which split mode is active for the
Braille display in use. See HJConst.jsh brlSplit\* constants for the
valid modes.

## Returns

Type: int\
Description: the active split mode.\

## Parameters

No Parameters

## Version

This function is available in the following releases:

1.  JAWS 25.0 and later
